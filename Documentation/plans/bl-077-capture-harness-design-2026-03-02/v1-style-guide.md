# v1 Style Guide

## Voice
- Directive and concise.
- Present tense action cues.
- Keep warnings actionable and single-step.

## Terminal Format
- Uppercase section banners: `== Preflight ==`.
- One-line key/value rows for runtime facts.
- Cue lines prefixed with timestamp: `[cue t=16s] ...`.

## Color Intent (for HTML preview only)
- Background: deep navy.
- Primary accent: cyan.
- Warning accent: amber.
- Error accent: coral.
