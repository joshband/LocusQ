# QA and Evidence

## Minimum Coverage
- Startup hydration path.
- Control interaction path (click/keyboard/wheel).
- Native function success/failure path.
- State persistence and restoration path.

## Evidence
- Host matrix entry updates.
- Repro logs and timestamps.
- Updated manual acceptance rows when applicable.
