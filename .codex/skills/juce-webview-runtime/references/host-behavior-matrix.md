# Host Behavior Matrix

## Goal
Track where behavior diverges between hosts and plugin formats.

## Required Fields
- Host name/version
- Plugin format (AU/VST3/standalone)
- OS and architecture
- Repro steps
- Observed vs expected behavior
- Workaround and fix status
